##### [容积率](https://baike.baidu.com/item/%E5%AE%B9%E7%A7%AF%E7%8E%87/1397682?fr=aladdin)
- 容积率是指一个小区的地上总建筑面积与净用地面积的比率。
- 容积率越低，居民的舒适度越高。
- 一般而言，19层以上住宅的容积率为2.4~4.5。
